The SFT Algorithm Over ZN

Learning and Coding Theory Exercise
This applet is an implementation of the SFT algorithm over ZN for finding the list of significant Fourier coefficients for some function f:ZN &rarr C.
The applet accepts as input an Integer N representing ZN, a threshold &tau, an error bound &delta in (0,1), ||f||∞ the infinity norm of f, ||f||2 the Euclidean norm of f, a constant coefficient for calculating delta and a constant coefficient for calculating mA and mB as described in the pseudo-code as described in this paper (page 52).
The applet also has access to a query function f, allowing it to ask for the f-values of a polynomial number of inputs (in log(N)) from ZN.
The querying can be done in two methods:
  • Interactive function query, user supplies f values on demand.
  • The user supplies an XML file describing the function, e.g. monoms and their coefficients.
