The SFT Algorithm Over ZN

Learning and Coding Theory Exercise
This applet is an implementation of the SFT algorithm over ZN for finding the list of significant Fourier coefficients for some function f:ZN &rarr C.
The applet accepts as input an Integer N representing ZN, a threshold &tau and an error bound &delta in (0,1). It also has access to a query function f, allowing it to ask for the f-values of a polynomial number of inputs from ZN.
The querying is enabled by three methods:
  • Interactive function query, user supplies f values on demand.
  • The user supplies an XML file with pairs of monoms and their coefficients.
  • The algorithm randomly selects f-values, i.e. some random function f.
